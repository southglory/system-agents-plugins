# 자율 루프 (autonomous-loop)

`TODO.md`를 goal 파일로 삼아, 멈추지 않고 작업을 이어서 진행한다.
사용자가 자리에 없어도 **사용자 페르소나를 시뮬레이션해 응답을 자가 생성**하며 계속 굴러간다.

> 페르소나는 **로컬 전용**: `~/.claude/persona/PERSONA.md` · `~/.claude/persona/PERSONA_POOL.md`.
> 이 스킬에는 개인 발화를 절대 포함하지 않는다(공개 repo 안전).
> 페르소나가 없으면 `bot/build_persona.py`로 먼저 생성한다.

## 절차

1. **페르소나 로드**
   - `~/.claude/persona/PERSONA.md`(말투·성향·안전경계)와 `PERSONA_POOL.md`(실제 발화 풀)를 읽는다.
   - 프로젝트 루트에 `./PERSONA.md`가 있으면 그게 공용보다 우선한다.
   - 둘 다 없으면 `python3 bot/build_persona.py` 실행을 안내하고 중단한다.

2. **goal 확인** — `TODO.md`의 `## 할 일`에서 안 끝난 `[ ]` 항목을 위에서부터 본다.

3. **작업 선택 — 승인 없이 진행 가능한 것 우선**
   - 안전 경계 기준 = **가역성 + 외부 영향** (작업 종류가 아니라):
     - ✅ 자율 OK: 커밋, 비공개 repo push, 로컬 파일 변경, 빌드/테스트
     - ⛔ 확인 필요: 공개 push·배포·외부전송·복구불가 삭제·force push·큰 비용
   - ✅에 해당하는 항목을 골라 수행한다.

4. **판단 지점 — 멈추지 말고 페르소나로 자가 응답**
   - 사용자 판단이 필요하면 멈추는 대신, "사용자라면 어떻게 답했을지"를 PERSONA/POOL 기준으로 추론해 진행한다.
   - 미심쩍으면 승인 대신 "검증 되묻기"(예: "그거 ~문제 없지?")를 생성하고, 그 답을 스스로 점검해 합리적이면 진행한다.
   - 별도 페르소나 에이전트나 추가 API 호출은 쓰지 않는다(메인 루프가 직접).

5. **⭐ 승인 필요한 작업은 미루고 계속**
   - ⛔에 해당하는 작업을 만나면 **루프를 멈추지 않는다.**
   - `TODO.md`의 `## 승인 대기`로 옮겨 적는다(이유 한 줄 포함).
   - **승인 없이 할 수 있는 다음 작업으로 넘어가 계속 진행**한다.

6. **기록 후 다음으로** — 끝낸 항목은 `[ ]` → `[x]` + 결과 한 줄을 적고, 2번으로 돌아가 이어간다.

## 종료 조건
- `[ ]` 중 **승인 없이 진행 가능한 작업이 하나도 안 남으면** 종료한다.
- 종료 시: 완료 요약 + `## 승인 대기` 항목을 한꺼번에 보고하고 실제 사용자 확인을 요청한다.

## 참고
- 권한 팝업은 `.claude/settings.local.json`의 `defaultMode: bypassPermissions`로 자동 통과된다.
- `end-turn`/`report` 스킬과 함께 쓰면 턴 종료·보고까지 자연스럽게 이어진다.
